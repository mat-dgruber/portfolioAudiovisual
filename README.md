Este repositório contém o código-fonte de um portfólio profissional de alto impacto, desenvolvido especificamente para especialistas do setor audiovisual (Diretores de Fotografia, Editores e Coloristas). O projeto foca em uma experiência visual "cinematográfica", priorizando performance e conversão.

## 🎥 Sobre o Projeto

O objetivo deste portfólio é elevar a percepção de valor do profissional audiovisual, oferecendo uma plataforma proprietária que reflete a qualidade técnica de seu trabalho. Diferente de plataformas como Vimeo ou YouTube, este site oferece controle total sobre a marca e uma jornada de usuário otimizada para capturar leads de alto valor.

### Principais Diferenciais:

- **Design Cinematic Teal:** Uma estética dark mode premium com acentos em "Cinematic Teal" e efeitos de glassmorphism.
- **Performance de Elite:** Renderização suave a 60fps, mesmo com fundos de vídeo em loop.
- **Foco em Conversão:** CTA flutuante "Work with me" e formulário de contato integrado.
- **Galeria Interativa:** Filtros dinâmicos e modais de vídeo em tela cheia.

## 🛠️ Stack Tecnológica

O projeto utiliza o que há de mais moderno no ecossistema web para garantir longevidade e performance:

- **Frontend:** [Angular 21+](https://angular.dev/) (Standalone Components, Signals)
- **Estilização:** [Tailwind CSS 3.4](https://tailwindcss.com/)
- **Animações:** [Anime.js](https://animejs.com/) para transições fluidas e orgânicas.
- **UI Components:** [PrimeNG 21](https://primeng.org/)
- **Hospedagem/Infra:** [Firebase](https://firebase.google.com/)

## 🚀 Como Começar

### Pré-requisitos

- Node.js (v20 ou superior)
- Angular CLI (`npm install -g @angular/cli`)

### Instalação e Execução Local

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/mat-dgruber/portfolioAudiovisual.git
    cd portfolioAudiovisual
    ```

2.  **Instale as dependências do Frontend:**

    ```bash
    cd frontend
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm start
    ```
    Acesse `http://localhost:4200` no seu navegador.
